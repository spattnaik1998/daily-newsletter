================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Port 8000 already in use"
  → Kill the process or edit api_server.py to use different port

Issue: "Port 3000 already in use"
  → Run: npm run dev -- -p 3001 (in frontend directory)

Issue: "Backend not responding in frontend"
  → Make sure python api_server.py is running in Terminal 1

Issue: "No newsletter showing"
  → Generate one first: python main.py or click Generate button

Issue: "ImportError: No module named"
  → Run: pip install -r requirements.txt again

Issue: "ANTHROPIC_API_KEY not found"
  → Check .env file has your API key (already set)
